Biography

Research

Biologist are continuously looking for automatic methods to reduce the human effort on tasks with large dataset. These methods should be fast and accurate simultaneously.

My current research involves analysis of whole rat brain slices. I am developing unique sets of deep learning and clustering based methods to detect and classify the cells and reveal previously unknown cellular differences in cells and detect homogeneity among cells in brain.

Deep Hierarchical Profiling


In this study a deep clustering pipeline is implemented for biomedical data discovery. For this purpose, a scattering network (ScatNet) is used to extract the deep translation invariant features of each cell. These deep features are used in a hierarchical clustering algorithm to find similar groups of cells. We have designed a Graphical User Interface (GUI) to visualize the clusters on top of the images to give the biologists the ability to do further analysis. This method identifies unknown cellular alterations among the cells.

Spectal Unmixing


In this project a two step linear sparse unmixing algorithm is implemented.

In the first step, an unsupervised sparse unmixing is implemented for bleedthrough correction which inferes the non-specific channels (endmembers) and the abundance of the endmembers from the source channel.

In the second step, user can correct the abundance matrix in a semi-supervised manner for chemical co-localization correction of signals.
